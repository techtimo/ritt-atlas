Über diesen Atlas

Der VDD Rittatlas zeigt aktuelle Distanzritte und Rittvorrat-Veranstaltungen in Deutschland auf einer interaktiven Karte und Tabelle.

Die Daten stammen aus der VDD-MediaWiki und werden regelmäßig automatisch auf Änderungen geprüft.

Datenschutzerklärung

Diese Seite setzt keine Cookies und erstellt keine Nutzerprofile.

Lokaler Speicher (localStorage): Folgende Daten werden lokal in Ihrem Browser gespeichert – es werden keine personenbezogenen Daten übertragen:

  • vdd_d, vdd_t, vdd_sa – Veranstaltungsdaten (offline-Cache, max. 7 Tage)
  • vdd_favs – Ihre Favoriten (Liste von Ritt-IDs)
  • vdd_push_sub – Push-Abonnement-Daten (nur bei aktivierten Benachrichtigungen)
  • vdd_push_prefs – Ihre Benachrichtigungseinstellungen
  • vdd_pending_prefs – Ausstehende Einstellungsänderungen (wird beim nächsten Online-Start synchronisiert)

Drittanbieter: Beim Laden der Seite werden Ressourcen von folgenden Diensten geladen, an die dabei Ihre IP-Adresse übermittelt wird:

  • basemap.de / BKG (sgx.geodatenzentrum.de) – Vektorkartenkacheln und Kartenstil
  • GitHub (github.io, api.github.com) – Hosting der Website und Prüfung des letzten Datenabgleichs
  • VDD MediaWiki – Veranstaltungsbilder und Dokumente (nur beim Öffnen einer Veranstaltung)

Push-Server (fly.io): Wenn Sie Push-Benachrichtigungen aktivieren, wird Ihr Browser-Abonnement (Push-Endpoint-URL sowie kryptographische Schlüssel) auf einem Server von fly.io gespeichert. Die Speicherung erfolgt auf europäischen Servern (Region: Frankfurt). Fly.io verarbeitet diese Daten gemäß ihrer Datenschutzerklärung. Wenn Sie keine Push-Benachrichtigungen aktivieren, werden keine Daten an fly.io übermittelt.

Standortdaten: Die Standortfunktion verarbeitet Ihren Standort ausschließlich lokal im Browser. Es werden keine Standortdaten übertragen.
